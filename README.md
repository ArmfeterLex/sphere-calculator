# Sphere Calculator

Модуль для консольных приложений на C#, вычисляющий площадь поверхности и объём шара по радиусу с округлением до двух знаков.

## Функциональность

- Ввод радиуса шара
- Вычисление площади поверхности: S = 4πR²
- Вычисление объёма: V = (4πR³)/3
- Округление результатов до двух знаков
- Вывод обоих результатов

## Пример работы

Введите радиус шара
5
Площадь шара 314.16; Объем шара 523.60

Введите радиус шара
2.5
Площадь шара 78.54; Объем шара 65.45
